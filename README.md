# 日程管理网页应用

这个网页应用允许用户通过简单的聊天界面和可视化日历来管理他们的日程安排。

## 功能特点

- 交互式日历，支持日、周、月视图
- 两栏布局：日历显示和聊天输入
- 通过自然语言输入或点击日历添加任务
- 基本的时间和任务信息解析
- 在日历和聊天界面中显示添加的任务
- 通过点击日历中的任务可以删除任务

## 如何使用

1. 打开index.html文件在浏览器中查看应用
2. 使用日历界面直接点击添加、编辑或删除事件
3. 在聊天框中输入自然语言描述（如"明天下午3点开会"）来添加任务
4. 查看日历中自动更新的任务安排
